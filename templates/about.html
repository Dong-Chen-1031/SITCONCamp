關於 SITCONCamp

這是一個使用 Flask 框架建立的基本 Web 應用程式。

技術架構

  • Flask - Python Web 框架
  • Bootstrap - 前端 CSS 框架
  • Jinja2 - 模板引擎
  • RESTful API - API 設計

功能特色

  • 響應式設計
  • 模板繼承
  • 錯誤處理
  • 表單處理
  • API 端點

專案結構

sitconcamp/
├── main.py          # 主要應用程式檔案
├── pyproject.toml   # 專案配置檔案
├── templates/       # HTML 模板目錄
│   ├── base.html    # 基礎模板
│   ├── index.html   # 首頁模板
│   └── ...
└── static/          # 靜態檔案目錄
    ├── css/         # CSS 檔案
    ├── js/          # JavaScript 檔案
    └── images/      # 圖片檔案
