關於 SITCONCamp

這是一個使用 Flask 框架和 Tailwind CSS 建立的現代化 Web 應用程式架構。

技術架構

  • Flask Python Web 框架
  • Tailwind CSS 實用優先的 CSS 框架
  • Jinja2 模板引擎
  • RESTful API API 設計

功能特色

  • 響應式設計
  • 模板繼承
  • 錯誤處理
  • 表單處理
  • API 端點

專案結構 

sitconcamp/
├── main.py                 # 主要應用程式檔案
├── config.py               # 配置檔案
├── pyproject.toml          # 專案設定檔
├── requirements.txt        # Python 依賴套件
├── .env.example           # 環境變數範例
├── README.md              # 專案說明文件
├── templates/             # HTML 模板目錄
│   ├── base.html          # 基礎模板
│   ├── index.html         # 首頁
│   ├── about.html         # 關於頁面
│   ├── contact.html       # 聯絡頁面
│   ├── contact_success.html # 聯絡成功頁面
│   ├── 404.html           # 404 錯誤頁面
│   └── 500.html           # 500 錯誤頁面
└── static/                # 靜態檔案目錄
    ├── css/
    │   └── style.css      # 自訂樣式
    ├── js/
    │   └── main.js        # JavaScript 功能
    └── images/            # 圖片檔案
